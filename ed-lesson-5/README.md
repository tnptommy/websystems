## What you already know

You already know these two commands:

```bash
ls              # list files in current folder
cat file.txt    # read and print a file
```

Today you will add 3 new skills on top of that:

| Skill | Symbol | What it does |
|-------|--------|--------------|
| Redirection | `>` `>>` | Save output to a file |
| Pipe | `\|` | Send output of one command into another |
| Wildcard | `*` `?` `[]` | Find files by pattern |
