## What this lesson covers

HTML and CSS are the two languages that make every webpage you visit.

- **HTML** defines the **structure** and **content** — headings, paragraphs, links, images
- **CSS** controls the **appearance** — colours, fonts, spacing, layout

You cannot style a page without HTML. You cannot structure a page without understanding what CSS targets. They always work together.
